## 🌟 What is HealthTrack AI?

**HealthTrack AI** is a fully client-side, AI-powered health dashboard that analyzes your **blood and urine test parameters** and generates:

- 🔬 **Disease Predictions** with probability scores across 25+ conditions
- 🥗 **Personalized Diet Plans** tailored to detected conditions  
- 💊 **Nutrition Supplement Recommendations** with clinical dosages
- 🏃 **Lifestyle & Exercise Guidance** for long-term recovery
- 📋 **Follow-up Action Plans** with doctor referral advice

> **🔒 100% Private** — All analysis runs locally in your browser. No data is sent to any server.

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Amarexist/blood-urine-analyser.git

# Navigate to project
cd blood-urine-analyser

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser 🎉

## 💡 How the AI Engine Works

The `localAnalyzer.ts` engine uses **evidence-based clinical thresholds** to:

1. **Flag abnormal values** — compares each parameter against gender/age-adjusted reference ranges
2. **Score disease probability** — multiple parameters contribute weighted scores to each condition
3. **Generate recommendations** — condition-specific diet, nutrition, lifestyle advice pulled from clinical guidelines
4. **Prioritise output** — diseases sorted by probability; high-severity conditions flagged first

```typescript
// Example: Diabetes detection logic
let diabetesProb = 0;
if (glucose >= 126) diabetesProb += 40;   // ADA fasting threshold
if (hba1c >= 6.5)  diabetesProb += 45;   // WHO diagnostic criteria
if (urineGlucose is positive) diabetesProb += 18;
if (symptoms include thirst/urination)    diabetesProb += 10;
```
